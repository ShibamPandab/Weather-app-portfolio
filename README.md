# 🌦️ Weather App

A modern and responsive Weather Application built using HTML, CSS, and JavaScript.  
It fetches real-time weather data using a weather API and displays it with a clean UI.

## ✨ Features
- 🔍 Search weather by city name
- 🌡️ Real-time temperature display
- ☁️ Weather condition with icons
- 💧 Humidity and wind speed
- 📅 5-day forecast
- ⏱️ Hourly forecast
- 🌗 Dynamic background based on weather
- 🔁 °C / °F toggle
- ⚠️ Error handling (invalid city, API issues)
- 📱 Fully responsive design

---

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- OpenWeatherMap API
